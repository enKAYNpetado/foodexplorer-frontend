# Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

### Front-end
- Axios
- LocalStorage
- ReactJS
- react-router-dom
- react-select
- react-input-mask
- react-icons
- styled-components
- swiper
- Vite
### Back-end 
- Autenticação
- API Restful
- Cors
- Deploy 
- Express
- JWT
- Jest
- Knex.js
- Middlewares
- Node.js
- PM2
- SQLite
- Testes automatizados
- Upload de imagens
- Variáveis de ambiente
